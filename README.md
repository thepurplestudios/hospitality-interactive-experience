
# Pasta Via — Hospitality Interactive Experience

> A premium, UI/UX-driven restaurant landing experience built to explore how design, motion, and frontend engineering come together in modern hospitality.

---

## Overview

This project is part of my portfolio — a **landing page for an imaginary restaurant brand: Pasta Via**.

But it’s not just a website.

It’s an attempt to answer:

> *What does a modern, premium hospitality experience look like on the web?*

This project explores:

* storytelling through UI
* interaction design in a hospitality context
* responsive frontend engineering
* and creating something that *feels* as good as it looks

Inspired by real-world hospitality brands, this is built as a **design + frontend case study**.

---

## Why Hospitality?

The hospitality industry is evolving fast.

Restaurants today are not just physical spaces — they are **digital brands**.

A website is often:

* the **first impression**
* the **menu**
* the **reservation system**
* and the **brand experience**

With tech becoming core to hospitality:

* seamless UX = better conversions
* strong visual identity = brand recall
* smooth interactions = perceived quality

This project explores how **design + frontend** can elevate that experience.

---

## UX Thinking (Mini Case Study)

The entire experience is designed around a simple flow:

> Discover → Explore → Personalize → Trust → Visit / Reserve

### Key UX decisions:

* **No clutter**
  → Clean, editorial layout keeps focus on experience

* **Glass navigation**
  → Floating, non-intrusive, always accessible

* **No hamburger menu**
  → Direct actions > hidden navigation

* **Reservation as modal**
  → Keeps user in flow, no page break

* **Motion philosophy**
  → *premium > flashy*
  → subtle, intentional, cinematic

* **Mobile-first adjustments**
  → icon-based navigation
  → stacked content
  → touch-friendly interactions

## Design System

This project is heavily design-driven.

### Visual Style

* luxury minimal
* editorial typography
* glassmorphism UI
* layered imagery
* cinematic composition
* soft, calm aesthetics

---

### Colors

* Background → soft cream
* Primary → deep olive / near-black
* Text → muted green tones
* Glass → translucent white layers

> Designed to feel *warm, premium, and edible* (yes, that matters in food UI)

---

### Typography

* Brand → expressive, handwritten feel
* Headings → editorial serif
* Body → readable serif

→ Balance between **luxury + comfort**

---

### Glassmorphism

Used intentionally for:

* navbar (glass pill)
* CTA buttons
* modal (reservation form)
* footer

Not just aesthetic — it creates:

* depth
* layering
* focus

---

### Motion & Animation

* Framer Motion powered
* smooth, cinematic transitions
* scroll-based reveal
* subtle hover interactions

Goal:

> Make the UI feel *alive, not noisy*

---

### Loader Experience

* designed as an extension of the hero
* same visual composition
* smooth transition into main UI

→ feels like the experience is *continuing*, not loading

---

### Images

All images are **AI-generated** using:

* Gemini Nano Banana
* ChatGPT

→ Custom visuals tailored to the exact design vision
